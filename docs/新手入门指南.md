## 1. HMNL 是什么？

### 🎯 一句话解释

**HMNL 是一个集直播、社区讨论、内容创作于一体的综合性在线平台。**

### 🏗️ 形象比喻

你可以把 HMNL 想象成一个"数字城市"：

```
┌─────────────────────────────────────────────────────────┐
│                    HMNL 数字城市                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   📺 直播广场          💬 社区论坛         ✍️ 创作工作室  │
│   ─────────          ──────────         ───────────    │
│   观看直播            发帖讨论           写故事/世界观   │
│   弹幕互动            评论交流           AI 辅助创作     │
│   关注主播            结交朋友           分享作品        │
│                                                         │
│   🎮 娱乐区            📱 个人中心         🔧 管理后台    │
│   ─────────          ──────────         ───────────    │

│   地图浏览            消息通知           数据统计       │
```

### 🌟 核心定位

| 维度 | 说明 |
|------|------|

## 3. 主要功能一览

### 🎪 八大核心功能模块

#### 3.1 直播频道 📺

**功能说明：** 观看和开启直播的核心区域

| 功能 | 说明 | 谁可以用 |
|------|------|----------|
| 浏览频道 | 查看所有正在直播的频道 | 所有人 |
| 观看直播 | 高清流畅的直播播放 | 所有人 |
| 弹幕互动 | 实时发送弹幕与主播交流 | 登录用户 |
| 点赞关注 | 为喜欢的频道点赞、关注 | 登录用户 |
| 开启直播 | 入驻主播可推流直播 | 入驻主播 |

**使用示例：**
```
主播端：使用 OBS 等软件 → 输入专属推流地址 → 开始直播
观众端：进入频道列表 → 选择感兴趣的直播间 → 观看 + 发弹幕
```

**常见头衔示例：**

**特点：**
- 一个用户可以有多个头衔
- 头衔以彩色徽章形式展示

## 6. 快速开始指南

### 🚀 5 分钟上手 HMNL

#### 第一步：访问网站

打开浏览器，访问：https://hmnl3.20110208.xyz

---

#### 第二步：注册账号

```
1. 点击右上角「登录/注册」按钮
2. 切换到「注册」标签
3. 填写以下信息：
   - 用户名（必填，用于登录和显示）
   - 密码（至少 6 位）
   - 图形验证码（不区分大小写）
4. 点击「注册」按钮
5. ✅ 注册成功！如果是第一个注册用户，自动成为超级管理员
```

**💡 小贴士：**
- 用户名一旦注册无法更改，请谨慎选择
- 建议使用邮箱作为用户名，方便记忆
- 密码建议包含字母和数字，更安全

---

#### 第三步：完善资料

```
1. 登录后，点击侧边栏「个人中心」
2. 点击「选择头像」上传头像照片
   - 支持 JPG、PNG 格式
   - 文件大小不超过 1MB
3. 填写个人简介（可选）
4. 点击「保存」按钮
```

**📸 头像建议：**
- 使用清晰的正面照片或卡通形象
- 避免使用侵权图片
- 头像会显示在你的帖子、评论、聊天消息中

---

## 7. 常见问题解答

### ❓ 注册登录类

**Q1: 注册时需要手机号吗？**
> A: 不需要。HMNL 支持纯用户名注册，手机号是可选的。但绑定手机号后可以使用短信验证码登录。

**Q2: 忘记密码怎么办？**
> A: 目前需要联系管理员重置密码。建议在设置中绑定手机号，未来将支持自助找回密码。

**Q3: 第一个注册用户为什么是超级管理员？**
> A: 这是 HMNL 的设计特色。第一个注册用户自动成为超级管理员，方便快速搭建和管理平台。如果是公开注册，建议先关闭注册，由管理员创建初始账号后再开放。

**Q4: 人脸识别登录怎么用？**
> A: 需要先在个人中心注册人脸信息。注册后，登录时选择「人脸登录」，按照提示完成识别即可。

---

### ❓ 直播功能类

**Q5: 如何成为入驻主播？**
> A: 
> 1. 完成实名认证（可选但推荐）
> 2. 联系管理员申请入驻
> 3. 管理员审核通过后，你会获得「入驻主播」头衔
> 4. 在个人中心查看专属推流地址

**Q6: 推流地址是什么？怎么用？**
> A: 
> - 推流地址格式：`rtmp://tv.20110208.xyz/live/你的用户 ID`
> - 使用方法：
>   1. 下载 OBS Studio（免费开源软件）
>   2. 打开 OBS → 设置 → 推流
>   3. 服务选择「自定义」
>   4. 填入推流地址和推流密钥
>   5. 添加画面源 → 开始推流

**Q7: 直播支持哪些格式？**
> A: 
> - 推流：RTMP 协议
> - 播放：HLS（.m3u8）和 FLV 两种格式
> - 分辨率：支持 480P、720P、1080P 等

**Q8: 直播收费吗？**
> A: 目前基础直播功能免费。未来可能推出高级功能（如更高画质、更多并发）的付费套餐。

### ❓ 社区功能类

**Q9: Markdown 是什么？怎么用？**
> A: Markdown 是一种轻量级标记语言，让你用简单的符号实现排版效果。

> 常用语法：
> ```
> # 一级标题
> ## 二级标题
> **加粗文字**
> *斜体文字*
> - 列表项
> [链接文字](网址)
> ![图片描述](图片 URL)
> ```

> HMNL 的编辑器支持实时预览，你可以在左侧写 Markdown，右侧即时看到效果。

**Q10: 帖子可以被删除吗？**
> A: 
> - 作者可以随时删除自己的帖子
> - 管理员可以删除任何违规帖子
> - 删除后不可恢复，请谨慎操作

**Q11: 如何获得更多头衔？**
> A: 头衔由管理员根据用户的贡献和表现授予。积极参与社区、创作优质内容、帮助他人都有机会获得头衔。

### ❓ 创作工具类

**Q12: 世界观创作是什么？**
> A: 这是一个专为虚构世界创作者设计的工具。你可以：
> - 构建完整的世界观设定（地理、历史、种族、魔法系统等）
> - 编写故事和角色背景
> - 自定义展示样式
> - 公开分享给其他用户

**Q13: 创作的内容版权归谁？**
> A: 版权归作者所有。HMNL 仅提供存储和展示平台。建议重要作品自行备份。

**Q14: 可以将作品设为私密吗？**
> A: 可以。创建世界观时可以选择「公开」或「私密」。私密作品只有你自己能看到。

### ❓ AI 功能类

**Q15: AI 对话需要额外付费吗？**
> A: 目前基础 AI 对话功能对所有登录用户免费开放。部分高级模型可能需要消耗虚拟货币（未来推出）。

**Q16: AI 能做什么？**
> A: AI 可以帮你：

> - 代码编写、调试帮助

> - 分析上传的文件

**Q17: AI 的回答准确吗？**
> A: AI 基于训练数据生成回答，大多数情况下是准确的，但也可能出现错误。重要信息请自行核实。

### ❓ 安全隐私类

**Q18: 我的个人信息安全吗？**
> A: HMNL 采取多重安全措施：
> - 密码加密存储
> - HTTPS 传输加密
> - 实名认证信息仅超管可见
> - 不向第三方出售用户数据

**Q19: 遇到骚扰或违规内容怎么办？**
> A: 
> 1. 使用举报功能举报违规内容
> 2. 拉黑骚扰用户
> 3. 联系管理员处理
> 4. 严重情况可报警处理

**Q20: 可以注销账号吗？**
> A: 可以。联系管理员申请注销账号。注销后，你的个人信息将被删除，但你发布的内容可能保留（已匿名化）。
